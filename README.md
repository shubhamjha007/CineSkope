# 🎬 CineSkope

CineSkope is a full-stack movie information web application that aggregates movie ratings from multiple platforms.

Users can search any movie and instantly view:

- 🎬 Movie Poster
- 📝 Overview
- ⭐ TMDb Rating
- ⭐ IMDb Rating
- 🍅 Rotten Tomatoes Rating

---

## Features

- Search any movie
- Real-time movie information
- Multi-platform ratings
- Clean dark-themed UI
- REST API based architecture
- Secure API key management using dotenv

---

## Installation

```bash
git clone <repository-url>

cd CineSkope/backend

npm install

npm start
```

Open

```
http://localhost:5000
```
